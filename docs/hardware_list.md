推荐硬件清单（针对 6-15 张台球桌）

1) 台桌检测与控制
- 灯控检测模块（每桌）
  - 方案 A：Shelly 1 / Shelly 1PM（干接点/继电器检测，支持 MQTT/HTTP），优点：易集成。估价：USD 8-20/台
  - 方案 B：定制 ESP32 + 光电隔离输入（读取灯的开关状态，通过 WiFi 上报），估价：USD 8-15/台（含外壳）
- 备用传感器（可选）
  - 垫子压力传感器或红外对射：用于冗余占用检测，估价：USD 10-40/台

2) 边缘网关与服务器
- 边缘计算机（1 台）
  - Raspberry Pi 4 (4GB/8GB)：USD 50-120（适合小规模）
  - 或 Intel NUC / 入门型商用小主机（更可靠，推荐用于商用）：USD 200-500
- 存储：建议 256GB NVMe 或 1TB SSD 备份（取决于报表保留策略）

3) 网络与基础设施
- PoE 交换机（若使用 PoE 摄像头/设备）：按端口计价，估价 USD 100-300
- 管理型 WiFi AP（覆盖整个厅）x2：USD 80-200/个
- UPS（关键设备供电，防断电）：USD 100-300

4) 收银与外设
- Android 平板或触屏终端（收银用）：USD 100-300/台（视品牌与规格）
- 热敏打印机（收据打印，USB/网络）：USD 50-150
- 二维码扫描器 / 摄像头（扫描顾客付款二维码）：USD 30-80
- 备用键盘 / 条码/会员卡读写器（可选）

5) 安全与安装
- 机架/外壳、布线材料、DIN 导轨盒、端子排、安装支架：合计 USD 50-200
- 小型网络机柜（1U/2U）及散热：USD 50-200

6) 可选（后期扩展）
- 摄像头 + 边缘 GPU（用于视觉分析、人数检测、球局录像）：USD 100-400/台（视型号）
- 专业 POS 设备（带刷卡及打印）：USD 300-800

采购建议
- 优先按“灯控检测 + 网关 + 本地服务器 + 平板收银”的最小组合采购，先做 1-2 张桌的现场试验（试运行 1-2 周）以验证误报率与网络稳定性。
- 若预算允许，使用 Intel NUC + 商用 PoE 交换机 + 企业级 AP，有利于长期稳定运行。

备注：以上价格为市场估算（USD），不同地区/供应商价格差异较大。针对马来西亚东马，建议联系本地电子零件分销商或使用 Lelong/Mudah 及淘宝/阿里采购对比价格，并考虑关税/运费。
